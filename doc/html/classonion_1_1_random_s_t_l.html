Detailed Description

template<class random_engine_t = std::minstd_rand>
class onion::RandomSTL< random_engine_t >

Implements the RandomEngine interface using the C++ STL facilities.

Parameters
random_engine_tSTL class used as the random number generator.

Note the random_engine is a template parameter because the STL does not structure its random engine classes on top of a commom base class. In other words: altough STL random engines do implement the same functions (what make it possible using them as template parameters), they do not share a base class and are not polymorphic objects.
